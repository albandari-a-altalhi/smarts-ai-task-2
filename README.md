# تصنيف الكائنات باستخدام OpenCV

## فكرة المشروع

هذا المشروع يطبق نظاما بسيطا لتصنيف الكائنات داخل الصور باستخدام Python و OpenCV. يعتمد الحل على تحليل الصورة واستخراج حدود الأجسام الظاهرة، ثم تصنيف كل جسم حسب شكله الهندسي ولونه الغالب.

النظام لا يحتاج إلى نموذج ذكاء اصطناعي مدرب مسبقا، بل يستخدم تقنيات معالجة الصور الكلاسيكية مثل تحويل الصورة إلى تدرج رمادي، تقليل الضوضاء، كشف الحواف، استخراج الكونتورات، وتحليل الخصائص الهندسية لكل جسم.

## ماذا يفعل البرنامج؟

- قراءة صورة من الجهاز.
- إنشاء صورة تجريبية تلقائيا عند عدم توفير صورة.
- اكتشاف الأجسام المنفصلة داخل الصورة.
- تصنيف الشكل إلى مثلث، مربع، مستطيل، دائرة، بيضاوي، خماسي، أو مضلع.
- تحديد اللون الغالب لكل جسم.
- رسم حدود ومربعات حول الأجسام المكتشفة.
- حفظ صورة نهائية موضح عليها التصنيف.
- تصدير النتائج في ملفات JSON و CSV.

## طريقة الحل

1. يتم تحميل الصورة باستخدام OpenCV.
2. يتم تحويل الصورة إلى Grayscale لتسهيل تحليل الحواف.
3. يتم تطبيق Gaussian Blur لتقليل الضوضاء وتحسين جودة الكشف.
4. يتم استخدام Canny Edge Detection لاستخراج الحواف الأساسية.
5. يتم تطبيق عمليات Morphology لإغلاق الفراغات الصغيرة داخل الحواف.
6. يتم استخراج الكونتورات الخارجية التي تمثل الأجسام المرشحة.
7. يتم تجاهل الأجسام الصغيرة جدا حسب قيمة `min-area`.
8. يتم تقريب شكل كل كونتور باستخدام `approxPolyDP`.
9. يتم تحديد نوع الشكل من عدد الرؤوس ونسبة العرض إلى الارتفاع ومعامل الاستدارة.
10. يتم تحويل الصورة إلى HSV وتحليل البكسلات داخل كل جسم لتحديد اللون الغالب.
11. يتم حفظ النتائج النهائية في مجلد `output`.

## المتطلبات

- Python 3.10 أو أحدث
- OpenCV
- NumPy

## التشغيل

إنشاء بيئة افتراضية:

```bash
python -m venv .venv
```

تفعيل البيئة على Windows:

```bash
.venv\Scripts\activate
```

تثبيت المكتبات:

```bash
pip install opencv-python numpy
```

تشغيل البرنامج بالصورة التجريبية:

```bash
python object_classifier.py --create-demo
```

تشغيل البرنامج على صورة مخصصة:

```bash
python object_classifier.py --input path/to/image.png --output output
```

تغيير أقل مساحة مسموحة للجسم:

```bash
python object_classifier.py --input path/to/image.png --min-area 2000
```

## المخرجات

بعد التشغيل سيتم إنشاء مجلد `output` يحتوي على:

- `annotated.png`: الصورة بعد رسم حدود الأجسام وكتابة التصنيف عليها.
- `results.json`: النتائج بشكل منظم مناسب للقراءة البرمجية.
- `results.csv`: النتائج في جدول يمكن فتحه ببرامج الجداول.

كل نتيجة تحتوي على:

- رقم الجسم.
- نوع الشكل.
- اللون الغالب.
- درجة ثقة تقريبية.
- مساحة الجسم.
- إحداثيات ومقاس المربع المحيط بالجسم.

## ملاحظات تقنية

هذا الحل مناسب للصور التي تحتوي على أجسام واضحة وخلفية بسيطة نسبيا. في الصور المعقدة جدا أو التي تحتوي على إضاءة ضعيفة وتداخل كبير بين الأجسام، قد تحتاج قيم المعالجة مثل `min-area` أو حدود كشف الحواف إلى تعديل حسب طبيعة الصورة.
